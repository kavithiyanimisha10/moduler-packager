# 🛠️ Multi-Utility Toolkit (Python)

A modular, menu-driven command-line application built in Python that provides multiple utility features such as datetime operations, mathematical calculations, random data generation, UUID creation, and file handling.

---

## 📦 Project Structure

```
Tools/tool.txt
│
├── main.py
│
├── Toopls/
│   ├── __init__.py
│   ├── datetime_tools.py
│   ├── math_tools.py
│   ├── random_tools.py
│   └── file_tools.py
│
└── README.md
```

---

## 🚀 Features

### 🕒 Datetime & Time Operations

* Display current date and time
* Calculate difference between two dates
* Format dates into custom formats
* Stopwatch (using high precision timer)
* Countdown timer

---

### 🧮 Mathematical Operations

* Factorial calculation
* Compound interest calculation
* Trigonometric functions (sin, cos, tan)
* Area of geometric shapes (circle, rectangle, triangle)

---

### 🎲 Random Data Generation

* Random number generator
* Random list generator
* Random password generator
* OTP generator

---

### 🆔 UUID Generation

* Generate unique identifiers using UUID v4

---

### 📁 File Operations

* Create a new file
* Write to a file
* Read file content
* Append data to a file

---

### 🔍 Module Exploration

* Explore attributes of any Python module using `dir()`

---

## ▶️ How to Run

1. Navigate to the project directory:

```bash
cd multi_toolkit
```

2. Run the main program:

```bash
python main.py
```
